# 📊 Excel Sales Dashboard

## SkillCraft Technology Internship - Task 01

This project was completed as part of the **SkillCraft Technology Internship Program**.

The objective of this project is to analyze Superstore sales data and create an interactive dashboard using Microsoft Excel.

---

## 📌 Project Objective

- Analyze sales performance.
- Create interactive dashboards.
- Visualize business insights.
- Practice Excel data analysis skills.

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

---

## 📈 Dashboard Features

✔ Total Sales KPI

✔ Total Orders KPI

✔ Total Customers KPI

✔ Average Sales KPI

✔ Sales by Category

✔ Sales by Region

✔ Sales by Segment

✔ Monthly Sales Trend

✔ Top 10 Products by Sales

✔ Interactive Slicers
